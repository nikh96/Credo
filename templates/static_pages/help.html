Help & FAQs

How do I add a transaction?

Go to Transactions → Add Transaction, fill in date, description, category and amount (positive for income, negative for expense), then save.

How do budgets work?

On the Budgets page, set a monthly limit per category. Credo compares that limit with expenses in the same category for the current month.

I forgot my password

On the login page, use “Forgot your password?” and follow the reset link shown in the server console.
